## 🚀 Funcionalidades & Soluções Técnicas

* **Portfólio Blindado de Vídeos:** Exibição imersiva com vídeos em *autoplay* e *loop* utilizando tags nativas do HTML5. Implementei proteções contra download não autorizado via CSS (`pointer-events: none`) e atributos HTML (`nodownload`, `oncontextmenu`).
* **Animações Zero-Dependências:** Elementos surgem suavemente conforme o scroll utilizando a API nativa `IntersectionObserver` do JavaScript, garantindo um visual dinâmico sem o peso de bibliotecas externas.
* **Tipografia Fluida:** Utilização de funções matemáticas nativas do CSS (`clamp()`) para que os textos se adaptem organicamente a qualquer tamanho de tela, otimizando a legibilidade e reduzindo *Media Queries*.
* **Funil de Conversão via WhatsApp:** Botão flutuante posicionado estrategicamente com animação de *pulse* contínua, desenhado para reduzir o atrito e acelerar o contato comercial.
* **Otimização Extrema de SEO:** Estrutura semântica rigorosa e meta tags (Open Graph) configuradas de ponta a ponta, resultando no Top 1 orgânico das buscas e garantindo um *preview* perfeito nas redes sociais.

**Arquitetura:** Projeto construído com arquitetura estática pura para garantir **velocidade de carregamento máxima** e ranqueamento superior, dispensando frameworks desnecessários.

**Destaques Técnicos:**
* Código estritamente modular (HTML, CSS e JS isolados).
* Gerenciamento de design system via Variáveis CSS (`:root`).
* Manipulação de DOM estritamente com **Vanilla JavaScript**.
* Alta performance de execução em dispositivos móveis.
