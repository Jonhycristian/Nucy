O projeto apresenta transformações de mídia com IA através de uma interface moderna, fluida e totalmente responsiva.

---

## 🚀 Funcionalidades

* **Portfólio Dinâmico de Vídeos**
  Exibição imersiva com vídeos em autoplay e loop utilizando HTML5, permitindo visualização instantânea das transformações.

* **Animações Inteligentes (Intersection Observer)**
  Elementos surgem suavemente conforme o scroll, garantindo dinamismo sem comprometer a performance.

* **Header Dinâmico & Menu Mobile**
  Navegação otimizada com menu responsivo e cabeçalho adaptativo durante a rolagem.

* **Conversão via WhatsApp**
  Botão flutuante com animação contínua, reduzindo atrito e facilitando o contato direto.

* **SEO e Social Preview**
  Meta Tags e Open Graph configurados para compartilhamento profissional em redes sociais.

**Arquitetura:**
Projeto estático com foco total em **velocidade, simplicidade e controle do código**.

**Diferenciais técnicos:**

* Código limpo e modular (HTML, CSS e JS separados)
* Uso de variáveis CSS (`:root`)
* Tipografia via Google Fonts (Inter)
* Ícones com Font Awesome
* Responsividade completa com Media Queries
